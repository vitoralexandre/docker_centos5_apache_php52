# Docker CentOS 6 with Apache and PHP 5.3

Docker with Apache 2.2 and PHP 5.3 executing suPHP. 

## Instructions 
- Create dir for web sites files: 
```
mkdir /virtual
```

- Create dir for template files: 
```
mkdir /opt/templates
```

- Create dir for Apache logs: 
```
mkdir /var/log/httpd
```

- Build Dockerfile 
```
docker build -t NAME . 
```
